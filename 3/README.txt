Создать шаблонную версию класса Image из предыдущей
лекции («Семинар 1.»). Необходимо как в примере с
«Шаблонный класс массива» сделать, чтобы Image хранил
внутри себя данные, фиксированного размера, выделенные
на стеке. Тип данных, также должен быть шаблонным.
Пример как это может выглядеть:
template<typename DataType, std::size_t mx,
std::size_t my>
class Image {
…
private:
DataType img[mxConst][myConst];
};
